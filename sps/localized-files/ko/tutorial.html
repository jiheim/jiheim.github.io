연습하기

커버하고 싶은 음악을 틀고 마음껏 연습해보세요. 녹음하기 전 사운드 이펙트를 조절하여 더욱 섬세하게 연습해볼 수 있습니다.

프로젝트 설정(곡 설정)

곡 제목, 가수명은 영상에 노출되니 꼭 수정해 주세요. 앨범 아트 이미지의 경우 비디오 영상이 없는 경우 해당 이미지로 대체되어 보입니다. 원하는 이미지를 순서대로 등록해보세요.

가사추가/입력/수정

가사를 직접 추가하거나 입력해보세요. 음원에 가사가 내장되어 있는 경우 자동으로 추가됩니다.

가사 추가/싱크 조정

화면에 가사를 추가하고 싶다면 가사에 싱크를 맞춰주세요. 음악에 맞춰 클릭하면 원하는 시간대로 싱크가 맞춰집니다. 싱크를 잘 못 맞췄어도 괜찮아요. ‘^’를 클릭하면 이전 가사를 다시 조정할 수 있습니다.

가사 싱크 상세 조정

가사별로 싱크 시간에 맞춰 상세 조정이 가능합니다.

오디오/비디오 리코딩

‘+’ 버튼을 누르면 오디오와 비디오를 선택하여 리코딩 할 수 있습니다.

클립 상세 조정

클립 선택 시 음량/음정/ 자동 볼륨/이퀄라이저 등 클립을 다양하게 조정할 수 있습니다.

클립자르기(Trim 기능)

클립이 중복되거나 필요 없는 부분이 생겼을 때 다시 녹음하지 마시고 자르기(Trim) 기능을 활용해보세요. 다중 녹음(녹화) 후 필요 없는 부분만 자를 수 있어 더 편하게 녹음(녹화)을 하실 수 있습니다.

클립 싱크 조정

리코딩된 클립이 기본 클립과 싱크가 맞지 않을 경우, 0.001초의 미세 단위로 조절할 수 있습니다.

비디오 템플릿

나의 영상에 맞는 템플릿을 선택해보세요. 한 번의 선택으로 멋진 영상을 만들 수 있습니다.

비디오/오디오 저장

비디오/오디오 등 원하는 형식으로 저장하실 수 있습니다. 또한 키네마스터 앱으로 내보내기를 통해 추가 편집도 가능합니다.

프로젝트 공유(키네마스터)

비디오 템플릿 적용 후 추가 편집이 필요하다면 ‘키네마스터’ 버튼을 눌러 공유해보세요. 영상을 다양하게 편집 하실 수 있습니다.

앱 언어 변경

싱플레이 정보에서 원하는 앱 언어로 변경하여 사용할 수 있습니다.

검색 기능

원하는 비디오/오디오 찾기 힘드시다면 검색 기능을 활용해보세요. 빠르고 쉽게 찾으실 수 있습니다.

원곡 미디어 파일 백업 기능 추가

앱에서 작업 중이던 원본파일이 지워져 걱정이시라고요? 이제 원본 파일이 지워져도 걱정하실 필요 없습니다. 설정 메뉴에서 백업을 선택하시거나, 원하는 프로젝트 설정 메뉴에서 개별로 백업하실 수 있습니다.

녹음 후 상세 싱크 조정

녹음이 지연된다면 녹음 후 확인 화면에서 바로 싱크를 조정해보세요. 확인 화면에서 바로 조정할 수 있어, 더 편하게 녹음하실 수 있습니다.

앱 설정

싱크 조정, 품질, 원본 파일 백업 ON/OFF 등의 기능을 설정 메뉴에서 적용하시면 별도로 설정하지 않아도 레코딩을 하면 바로 적용됩니다.
